# Minimal App
This app is just a minimal example with interaction.

## Running
Enter 
```
$ bokeh-server --script minimal_app.py
```
in bash to run the app. Then enter
```
http://localhost:5006/bokeh/minimal
```
in your browser to use the app in it.
